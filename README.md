# TE

A simple example of React with Atomic design.

[Demo](https://adrienloup.github.io/te/)

## Requirements

- `node` : `^18.18.2`
- `npm` : `^9.8.1`

## Installation

```bash
nvm use
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Checking Code Quality

```bash
npm run lint
npm run lint:fix
```

## Checking Code Tests

```bash
npm run cy
npm run cy:open
```
